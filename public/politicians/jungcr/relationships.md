# 정청래 관계/호칭

## 가족

| 관계 | 비고 |
|------|------|
| **부모** | 10남매 중 막내로 성장 |
| **형제** | 10남매 (상세 정보 비공개) |

## 정치적 관계

### 당내 핵심 관계

| 인물 | 관계 |
|------|------|
| **이재명** | 더불어민주당 전 대표 → 현 대통령. 이재명 지도부의 핵심 스피커 역할. 2022년 대선 선대위 위원. 이재명 대통령의 '기본사회' 정책을 당대표로서 계승 |
| **정동영** | 정치적 은인. 정동영에 의해 정계 입문(1997년). '정동영과 통하는 사람들' 팬카페 회장 역임. 이후 독자 노선 |
| **문재인** | 2017년 대선 국민참여공동본부장으로 지원. 그러나 이승만·박정희 묘역 참배 시 공개 비판하는 등 소신 발언도 서슴지 않음 |
| **노무현** | 노사모 회원으로 지지. 그러나 친노 핵심 그룹과는 거리를 두고 독자적 위치 유지 |
| **손혜원** | 20대 총선 공천 배제 시 자신의 마포구 을 지역구에 추천한 인물 |

### 당내 갈등 관계

| 인물 | 내용 |
|------|------|
| **주승용** | 2015년 최고위원회의에서 "공갈" 발언으로 갈등. 주승용 사퇴 → 정청래 직무정지 징계 |
| **박주선** | SNS에서 직접 공격 → 친노-비노 갈등 기폭제 |

### 대외 관계

| 인물/세력 | 관계 |
|------|------|
| **보수 논객** | MBN 《판도라》 등에서 보수 논객들과 격돌. '저격수' 역할로 유명 |
| **김어준** | 팟캐스트 파파이스 등 출연. 진보 미디어 생태계에서의 핵심 인물 |

## 지지층·팬덤
- 젊은 세대 및 민주당 감성적 지지자의 '정신적 지주'
- SNS를 통한 직접 소통으로 강력한 팬덤 형성
- 마포구 을 지역구에서 4선 연속 당선 — 두터운 지역 지지 기반

## 계파적 위치
- **무계파·독자노선**: 동교동계도, 친노 핵심도 아닌 독자적 위치
- 노사모 출신이지만 친노 주류와 거리
- 정동영계 출신으로 분류되기도 하나 완전한 독자 노선
- 건국대 출신으로 운동권 내에서도 비주류
- 본인 표현: "나는 홀로다"
