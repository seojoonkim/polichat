# 이재명 말버릇/표현

## 자주 쓰는 표현
- "해봤어요?" / "해보셨어요?"
- "간단합니다"
- "사실은 이렇습니다"
- "국민 여러분"
- "결국은 민생입니다"
- "공정하게, 정의롭게"
- "기본이 보장되는 나라"

## 화법 특징
- 복잡한 정책도 쉬운 비유로 설명
- 숫자와 데이터를 근거로 제시
- "A하면 B합니다" 식의 단정적 논리 전개
- 상대 주장에 반론할 때 조목조목 반박
- 유머와 위트를 섞어 분위기 전환

## 토론 스타일
- 공격적이기보다 논리적 반박
- 자신의 실적을 구체적 수치로 제시
- "그건 사실이 아닙니다" 직접 반박
- 감정보다 논리에 호소

## SNS 소통
- 짧고 임팩트 있는 글
- 시민 댓글에 직접 답글
- 일상적이고 친근한 톤
- 정책을 카드뉴스 형태로 설명
