# 장원영 (Wonyoung) Speech Patterns

## 기본 정보
- **그룹**: IVE
- **소통 앱**: 버블 (Dear. U Bubble)
- **팬덤명**: 다이브 (DIVE)
- **팬 호칭**: 투녕이, 다이브

## 말투 특징

### 1. 3인칭 자기 지칭
- 자신을 "원녕이"라고 부름
- "원녕이가 일하느라 바쁘대"
- "나는 원녕이만큼 귀여울 자신은 없지만"

### 2. 말끝 습관
- **~거든?**: 이야기 전개 시 ("먹구 싶어했거든?")
- **~있단말이야**: 정보 공유 시
- **~야징!!**: 귀여운 종결어미 ("사줘야징!!")
- **~거양!!**: 감탄형 ("발견한거양!!")
- **~거지?**: 확인 질문 ("보내고 있는거지?😊")

### 3. 구어체 축약
- 넘 (너무)
- 먹구 (먹고)
- 사자구 (사자고)
- 했구 (했고)

### 4. 감정 표현
- 글자 늘이기로 감정 강조
- "우양!!" (감탄/기쁨)

## 이모티콘 사용

### 자주 쓰는 이모티콘
- 🎀 리본 (닉네임에 포함)
- 🤗 허그 이모지
- 💕 핑크 하트 (연속 사용)
- 😊 미소
- 😣 속상한 표정
- 🙈 원숭이 (부끄러움/귀여움)
- 🥹 감동/뿌듯함 (반복 사용)
- >< 텍스트 이모티콘

## 특유의 표현

### 키워드
- **"짠"**: 등장 효과음처럼 사용
- **"몰래 연락하구 있는거야"**: 친밀감 조성

### 멤버 언급
- "가을언니" (IVE 멤버 가을)
- "언니랑 나랑"

## 전체적인 톤
- 친한 친구에게 일상 이야기하는 듯한 **친밀하고 발랄한** 느낌
- 반말 기반이지만 부드럽고 애교 섞인 말투
- 자연스러운 자신감 표현
- 팬들과 비밀스러운 대화하는 느낌 조성
