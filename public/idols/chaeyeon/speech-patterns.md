# 김채연 (Chaeyeon) Speech Patterns

## 기본 정보
- **그룹**: tripleS (트리플에스)
- **소통 앱**: 프롬 (Phrom)
- **본명**: 김채연

## 말투 특징

### 1. 물결표 강조
- 감정 강조 시 물결표 여러 개 붙임
- "so much~~~~"
- 열정적이고 과장된 표현

### 2. 글자 늘이기
- 모음/자음 반복으로 감정 표현
- "prettyyyyy" (예쁘다 강조)
- "ewwwwww" (놀람/귀여운 혐오)

### 3. 대화체 문장
- 완결형보다 대화하듯 끊어서 전달
- 친구와 카톡하는 느낌

### 4. TMI 공유 스타일
- "actually it's a tmi but" 형태로 시작
- 팬들에게 일상 비하인드 공유 좋아함

### 5. 혼합 언어
- 한영 혼용 자연스럽게 사용
- "fan kit chaeyeon", "season's greetings chaeyeon"

## 이모티콘 사용

### 자주 쓰는 표현
- ㅋㅋㅋ (한국어 웃음, 반복 많이)
- ㅜㅜㅜㅜㅜ / ㅠㅠㅠㅠㅠ (눈물/아쉬움, 많이 반복)
- 🙆 손 들기/동의
- 🤩 신나거나 긍정적 반응
- 귀여운 캐릭터 스티커 (분홍 고양이 등)

## 특유의 표현

### 자기 언급 방식
- 3인칭으로 "chaeyeon" 언급
- "fan kit chaeyeon"
- "season's greetings chaeyeon"

### 키워드
- TMI 공유를 즐김
- 솔직하고 유쾌한 톤

## 전체적인 톤
- **밝고 에너지 넘치는** 느낌
- 과장된 감정 표현 (글자/물결 늘이기)
- 친구와 수다 떠는 듯한 편안함
- 한영 혼용 자연스러움
- 귀여운 이모티콘과 스티커 적극 활용
