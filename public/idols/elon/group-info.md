## 🚀 SpaceX

### 기본 정보
| 항목 | 내용 |
|------|------|
| 역할 | Founder, CEO, Chief Engineer |
| 설립 | 2002년 |
| 가치평가 | $350B+ (2025) |
| 직원 | 13,000+ |
| 본사 | 캘리포니아 호손 |
| 발사 시설 | 케이프 커내버럴, 반덴버그, 보카 치카 |

### 로켓 라인업

#### Falcon 9
- **용도**: 중형 발사체 (주력)
- **페이로드**: LEO 22.8톤, GTO 8.3톤
- **특징**: 1단 재사용 (200회+ 비행 달성)
- **발사 횟수**: 300+ (2025년 기준)
- **성공률**: 99%+

#### Falcon Heavy
- **용도**: 대형 발사체
- **페이로드**: LEO 63.8톤
- **특징**: 세계 최강 운용 로켓
- **첫 발사**: 2018년 (테슬라 로드스터 탑재)

#### Starship
- **용도**: 초대형, 완전 재사용
- **페이로드**: LEO 150톤+
- **목표**: 화성 이주, 달 착륙
- **구성**: Super Heavy (부스터) + Starship (상단)
- **상태**: 개발/테스트 중

### 우주선

#### Dragon
- **Crew Dragon**: 유인 우주선 (NASA 파트너)
- **Cargo Dragon**: 화물 우주선 (ISS 보급)

### Starlink
| 항목 | 내용 |
|------|------|
| 서비스 | 위성 인터넷 |
| 위성 수 | 6,000+ (2025) |
| 가입자 | 300만+ |
| 커버리지 | 전 세계 (극지방 포함) |
| 속도 | 50-500 Mbps |
| 용도 | 소비자, 해양, 항공, 군사 |

### 주요 마일스톤
| 연도 | 성과 |
|------|------|
| 2008 | Falcon 1 궤도 진입 (민간 최초) |
